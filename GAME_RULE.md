# 게임 규칙: Fruit Catcher (과일 받기 게임)

> 하늘에서 떨어지는 과일을 바구니로 받아내는 포즈 인식 게임입니다!

## 🎮 게임 제목
**Fruit Catcher AI**

---

## 📝 게임 설명
화면 상단에서 다양한 과일과 폭탄이 떨어집니다.
플레이어는 카메라 앞에서 몸을 **왼쪽, 가운데, 오른쪽**으로 움직여 바구니를 이동시킵니다.
폭탄을 피하고 맛있는 과일만 골라 담으세요!

---

## 🎯 게임 목표
- 제한 시간(60초) 동안 최대한 많은 점수를 획득하세요.
- 폭탄을 3번 맞으면 게임이 즉시 종료됩니다.

---

## 🕹️ 게임 규칙

### 기본 조작
화면은 **3개의 레인(Lane)**으로 나뉩니다.
플레이어의 포즈에 따라 바구니가 해당 레인으로 이동합니다.

| 포즈 (Pose) | 바구니 위치 | 설명 |
|:---:|:---:|---|
| **Left** | 왼쪽 레인 | 왼쪽으로 고개를 기울이거나 몸을 움직이세요 |
| **Center** | 가운데 레인 | 정면을 바라보거나 차려 자세를 유지하세요 |
| **Right** | 오른쪽 레인 | 오른쪽으로 고개를 기울이거나 몸을 움직이세요 |

### 아이템 종류

| 아이콘 | 이름 | 점수 | 효과 |
|:---:|:---:|:---:|---|
| 🍎 | **사과** | +100점 | 기본 점수 획득 |
| 🍇 | **포도** | +300점 | 빠르게 떨어짐 (보너스) |
| 💣 | **폭탄** | **-1 Life** | **3번 맞으면 GAME OVER** |

### 게임 진행
1. 게임이 시작되면 과일과 폭탄이 무작위 레인에서 생성되어 떨어집니다.
2. 바구니가 과일에 닿으면 점수가 올라갑니다.
3. 바구니가 폭탄에 닿으면 생명이 줄어듭니다.
4. 시간이 다 되거나 생명이 0이 되면 게임이 끝납니다.

---

## 🎨 필요한 포즈 목록 (Teachable Machine 학습용)

Teachable Machine에서 아래 3가지 클래스를 학습시켜주세요.

1. **Left** (왼쪽)
2. **Right** (오른쪽)
3. **Center** (가운데/정면)

> **팁**: '가운데' 포즈는 아무것도 안 하고 가만히 서 있는 상태로 학습시키면 좋습니다.

---

## 📊 레벨 디자인 (선택 사항)

- **시간**: 60초
- **난이도**: 시간이 지날수록 과일이 떨어지는 속도가 점점 빨라집니다.
- **피버 타임**: 콤보 10회 달성 시 5초간 폭탄이 나오지 않고 점수 2배!

---

## 💡 개발 가이드 (AI 참고용)

### 구현 로직
1. **Lane System**: `x` 좌표를 3분할하여 관리 (예: 200px 캔버스라면 33px, 100px, 166px 중심)
2. **Falling Logic**: `requestAnimationFrame`을 사용하여 Y좌표 증가
3. **Collision**: 바구니의 Lane과 아이템의 Lane이 일치하고, Y좌표가 바구니 높이(하단)에 도달했을 때 충돌 판정

### 메타데이터 라벨 미리보기 (`metadata.json`)
```json
{
  "labels": ["Left", "Center", "Right"]
}
```
위 라벨과 정확히 일치하도록 모델을 학습해야 합니다.
